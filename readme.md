# 🛒 Blinkit Clone

A modern and responsive Blinkit-inspired grocery delivery landing page built using **HTML**, **CSS**, and **JavaScript**.

## ✨ Features

- Modern Blinkit-style UI
- Responsive design for mobile, tablet, and desktop
- Sticky navigation bar
- Search bar with modern styling
- Category cards with Font Awesome icons
- Animated offer banner
- Smooth hover effects
- Clean and maintainable code structure

## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/blinkit-clone.git
```

### 2. Navigate to Project Folder

```bash
cd blinkit-clone
```

### 3. Open Project

Simply open:

```bash
index.html
```

in your browser.

No installation required.
